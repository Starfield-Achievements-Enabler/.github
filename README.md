# Starfield Achievement Enabler

Starfield Achievement Enabler is a standalone tool designed to restore and enable achievements while using mods. It works by applying safe, reversible patches without permanently modifying your game files.

## Features

- **Standalone EXE**: No additional dependencies (e.g., Python or .NET) required — simply run `achievement_enabler.exe`.
- **Achievement Restoration**: Re-enables achievements that are normally disabled when using mods in Starfield.
- **Safe Overlay System**: Your game files remain unmodified — all changes are applied through a safe method.
- **Automatic Backup**: Critical game files are backed up before any changes are applied.
- **One-Click Enable/Disable**: Easily toggle achievement support on or off.
- **Mod-Friendly**: Designed to work alongside most Starfield mods without conflicts.
- **Lightweight & Fast**: Minimal performance impact on the game.
- **Cross-Platform Support**: Works on Windows with no external dependencies.

## Installation

### Windows
1. Download the latest release from the Releases page.
2. Extract the `.zip` file to your preferred directory.
3. Run `achievement_enabler.exe`.

---

## Usage

1. Launch Starfield Achievement Enabler.
2. Add your Starfield game directory via **Settings > Game Directory** (auto-detects Steam installs).
3. Click **ENABLE ACHIEVEMENTS** to apply the patch.
4. Launch the game normally.
5. Earn achievements even while using mods.

---

## Disabling / Restoring

- Click **DISABLE** to remove the patch and return to default behavior.
- Use **RESTORE BACKUP** to revert any modified files to their original state.

---

## Notes

- This tool does **not** modify your save files.
- Always keep backups to avoid potential issues after game updates.
- Compatibility may vary with major Starfield patches.
